6.5.4标准字符串处理函数字符串输入输出函数正注意函数存在设计缺陷,当读入的字符串长度大于存储它的字符数组(3) fgets 函数长度时会发生数组越界 。引发难以预的错误。原型:char* fgets(char* $, int n, FIIE* file);功能:  从文件流中读取一行数据 (遇到换行符 In'或输入文件的结尾 EOF ) 或指定个数的字符第个参数 $是保存读入字符串的字符型数组;  第二个参数n是读入的字符数约束;  第三个参数 file 表示读取的输入来源文件调用时如果将 stdin 传递给 file 则表示从标准输入读取至多读取 n-1 个字符 (加上自动添加的字符串结束符,最多将1个字符写入 $,防止越界)。如果输入的一行数据中包含换行符在内的字符数小于等于n-1,则全部读入 (包括换行符) ;否则只读入输入数据中前n-1 个字符。(4) tuts 函数原型:int puts(const char* $);功能:  将字符串8输出到文件 file 中,但不会在字符串最后再额外输出换行符。调用时如果将 stdout 传递给 file ,则表示从标准输入读取。gets
