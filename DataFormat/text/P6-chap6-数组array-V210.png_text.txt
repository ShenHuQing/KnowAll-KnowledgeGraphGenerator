6.1.2 一维数组初始化例 6-1 一维数组初始化例6-1代码分析1 int a[5] ={1,2,3,4,5};第1行,初始化列表中值的个数等于数组长度。数组元素2 int b[5] = {9,10};按次序被赋初值为初始化列表中的对应值。3 it c[5] = {11,12,13,14,15,16};第2行,初始化列表中值的个数小于数组长度。数组前24 int d[] = {21,22,23,24,25};个元素被赋初值9和10,其它元素初始化为0。5 int e[5] ={0};/1等价于 it e[5] ={};第3行,初始化列表中值的个数大于数组长度,只取初始化列表中前5 个值按次序赋值给数组 c的每个元素,此时6 int f[5];绝大多数编译器会给出一个警告指示出这个潜在的风险。第4行,定义时数组 d的长度没有显式给出,由后面初始化列表中值的个数确定,因此数组 d的长度为5,它每个元素依次初始化为21到25。第5行,数组 e的所有元素被初始化为0。第6行,定义数组f时没有初始化,因此f的元素值不确定;是随机的。
